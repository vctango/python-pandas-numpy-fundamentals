# Python Data Analysis Fundamentals with NumPy & Pandas

This repository contains hands-on Python projects completed while studying data analysis with NumPy and Pandas.

The projects focus on data manipulation, analysis, filtering, aggregation, HTML data ingestion, sorting, merging, and business-oriented data processing scenarios.

## Technologies Used

- Python
- NumPy
- Pandas
- Jupyter Notebook

## Repository Contents

| File | Purpose |
|--------|---------|
| `01_portfolio_analysis.py` | Stock portfolio valuation and financial data analysis |
| `02_retirement_age_html_parser.py` | HTML table ingestion and retirement age data analysis |
| `03_bank_customer_analysis.py` | Customer filtering, aggregation, and business analytics |
| `04_customer_data_merge.py` | DataFrame concatenation, merging, and customer master data management |

Detailed project descriptions, implementation notes, and code comments are included within each Python file.
